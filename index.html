Some Good Ideas

  • Strive for simplicity. If it's complicated to build, it's likely hard to use and maintain.
  • Help each other improve. We learn faster together than apart.
  • Don't the reinvent the wheel. Use the tools we already have when it makes sense.
  • Prioritize performance. You may not need all that JavaScript and CSS ;)
  • Think big picture. Is what you're doing really going to make an impact? If not, change course!
  • Use your products. If you don't like how something works, speak up.
